# Vertex Gym ERP

A full-stack starter Gym ERP built with React + Vite + Tailwind CSS, Node.js + Express, and MySQL.

## Modules Included
- Authentication with JWT
- Dashboard summary APIs
- Member management
- Trainer management
- Membership plans
- Attendance tracking
- Payment tracking
- Role-based routing scaffold
- MySQL schema and seed SQL

## Tech Stack
- Frontend: React, Vite, Tailwind CSS, React Router, Axios
- Backend: Express, MySQL2, JWT, bcryptjs, express-validator
- Database: MySQL

## Quick Start

### 1) Backend
```bash
cd backend
npm install
npm run dev
```

### 2) Frontend
```bash
cd frontend
npm install
npm run dev
```

### 3) Database
Import:
- `backend/src/database/schema.sql`
- `backend/src/database/seed.sql`
